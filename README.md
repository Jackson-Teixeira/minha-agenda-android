# 📱 Minha Agenda Android

Um aplicativo simples de agenda para Android desenvolvido em Kotlin.

## 🛠️ Funcionalidades
- Adicionar, editar e excluir contatos.
- Listagem de contatos em tempo real.
- Busca por nome ou telefone.
